Mathematics of Computer Vision

Python, OpenCV, Numpy, Pandas, Latex, Matlab
This project is an educational resource for learning Computer Vision consisting of 60+ pages of TexEd up notes and 10 Python Demos (Documented Jupyter Notebooks). These notes were presented at various FTC robotics conferences. Some relevant topics covered include the PnP Model, object detection, facial recognition, digital signals processing, image compression, and CNNs/Deep-Learning. Please read the README on the Github Repository before the notes!
TLDR: textbook-style notes written in Latex about CV + demos

Sparsity

C, C++, Python
This project contains implementations of Compressed Sparse Row Matrix representation (CSR), various matrix methods (multiplication, QR/LU/Cholesky Decomposition, etc), a Collaborative-Filtering scheme using my implementation of CSR (prediction scheme for content used by platforms like Netflix). It's implemented in C and has comparable run-times to similar functionalities of scipy.sparse, Python's standard module for sparse matrices. It is inspired by Dr. Bahar Asgari's PhD Thesis.
TLDR: library written in C wrapped with Python for sparse data

Surveillance Detector

Python, YOLO, PyTorch, TKinter
Human Detector uses your webcam to monitor for unauthorized access of your computer. When a person is detected, it prompts for a password and, if not entered, triggers an alarm, takes a photo, and can initiate other configurable actions like shutting down the computer. It uses a trained-version of the YOLO Object Detection System and contains thoroughly documented code on use/design of the YOLO Architecture!
TLDR: human security detector with alarm, password, and other features

Vehicle Type

Python, YOLO, PyTorch, Scikit-Learn, Pandas, Numpy
This project is a vehicle-classifier which classifies vehicles into multiple different categories based on images. It will be used in the future to analyze videos of vehicles and assess their speed and construct a "safety-score" on how the drivers are supposedly driving. It is intended to be a deep-learning based tools for traffic-cameras to learn and comment more about vehicles.
TLDR: vehicle detector for analyzing cars/vechiles and traffic cameras
